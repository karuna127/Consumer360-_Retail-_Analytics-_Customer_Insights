# Consumer360-_Retail-_Analytics-_Customer_Insights
Built a retail analytics project using SQL, Python, and Power BI to analyze customer behavior and sales performance. Applied data cleaning, revenue analysis, and segmentation techniques to identify top customers, best-selling products, and business growth insights.

Project Overview

This project focuses on analyzing retail sales data to understand customer behavior, product performance, and revenue trends. The goal is to help businesses make better decisions using data.

Tools Used
SQL (PostgreSQL) – for data cleaning and querying
Python (Pandas) – for data analysis and processing
Power BI – for dashboard and visualization

What I Did
Cleaned raw sales data (removed null and negative values)
Created revenue column
Analyzed:
Monthly sales trend
Top products
Revenue by country
Exported clean data using Python
Built an interactive dashboard in Power BI

Key Insights
Identified top revenue-generating countries
Found best-selling products
Observed sales trends over time
Helped understand customer purchasing patterns
